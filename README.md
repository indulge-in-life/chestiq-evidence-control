# Chestiq Evidence Control

A structured artefact control system for managing data, models, evaluations, and prompts.

## Repository Structure

- data/ → Input artefacts
- models/ → Model lifecycle assets
- evaluations/ → Metrics and validations
- prompts/ → LLM prompt templates
- docs/ → System documentation

## Principles

- Reproducibility
- Traceability
- Controlled artefact evolution
